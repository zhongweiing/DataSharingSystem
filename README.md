# JSP资料共享平台

#### 介绍
利用JSP实现的一个共享资料平台，大体上的功能有用户进行功能上传，然后上传资源的图片描述和文字描述，以及所需要的积分点。管理员对用户上传的资源进行审核，未审核的资源不会在页面展示，只有当管理员审核通过才会被展示。系统中用到了伪静态网站技术、ajax技术以及文件上传和下载。
